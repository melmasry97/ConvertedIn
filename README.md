# ConvertedIn Assessment  

## Installation

### 1- Run this command to start the application.

```
php artisan start:app
```

### 2- run this command to update statistics .

```
php artisan queue:work
```

### 3- then run tests.

```
php artisan test
```

Then go to 

```
 http://127.0.0.1:8000
```
